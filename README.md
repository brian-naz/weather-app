# Weather App

To get started:
<a href="https://weather-app-five-amber.vercel.app/">Demo</link>

### Setup:

Clone the repo:
```bash
git clone https://github.com/brian-naz/weather-app.git
```
Go to the directory and simply type:
```bash
npm install
```
To run:
```bash
npm start
```
